# websocket-react
## Requirements
```
node --version v16.13.1
npm --version 8.1.2
npx --version 8.1.2
```

## Setup
先に```https://github.com/YusakuSekine-Relic/websocket-server```のセットアップをお願い致します
```
cp .env.example .env.local
npm install
npm run start

open localhost:3000/client ← コメントが送れるページ
open localhost:3000/streamer ← コメントが流れてくるページ
```
